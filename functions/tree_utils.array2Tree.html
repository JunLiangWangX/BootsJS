Function array2Tree

  • array2Tree(arr, nodeIDAttributeName, parentIDAttributeName, generateChildListAttributeName, judgeRootNodeFunction): any[]

  • Convert array to tree.(把数组转换为树)

    Parameters

    • arr: any[]

      Given an array.(给定数组)

    • nodeIDAttributeName: string

      Attribute name of node ID.(节点ID的属性名称)

    • parentIDAttributeName: string

      Attribute name of parent node ID.(父节点ID的属性名称)

    • generateChildListAttributeName: string

      Specify the attribute name of the child node in the tree.(指定树中存放子节点的属性名)

    • judgeRootNodeFunction: ((node) => boolean)

      Function to determine whether a node is the root node.(判断节点是否为根节点的方法)

        • (node): boolean

        • Parameters

          • node: any

          Returns boolean

    Returns any[]

    Example

    const  TreeUtils  = require('boots-js/cjs/tree-utils'); // CommandJS
    import TreeUtils  from 'boots-js/tree-utils' // Es6 Module

       const arr = [
           { name: '中国', code: '0' , level:0 },
           { name: '重庆', code: '01', level:1 , parentCode: '0' },
           { name: '四川', code: '02', level:1 , parentCode: '0' },
           { name: '广东', code: '03', level:1 , parentCode: '0' },
       ]
       let genTree = TreeUtils.array2Tree(arr, 'code', 'parentCode', 'childList', (node) => {
           return !('parentCode' in node)
       })
       console.info(genTree)
