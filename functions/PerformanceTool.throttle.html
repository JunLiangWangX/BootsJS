Function throttle

  • throttle(func, delay?): Function

  • Throttle function.(节流函数)

    Parameters

    • func: Function

      Functions that require debounce.(需要防抖的函数)

    • delay: number = 300

      Delay，default 300ms.(延迟时间，默认300ms)

    Returns Function

    Example

    const  PerformanceTool  = require('boots-js/performance-tool'); // Node
    import * as PerformanceTool  from 'boots-js/performance-tool' // Es6 Module

    let num=0
    function add(nv){
       num+=nv
    }
    const throttleAdd=PerformanceTool.throttle(add);
    throttleAdd(1);
    throttleAdd(1);
    throttleAdd(1);
    console.info(num); // 1
