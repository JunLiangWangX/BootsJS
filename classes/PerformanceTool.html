Static debounce

  • debounce(func, delay?, isImmediate?): Function

  • Debounce function.(防抖函数)

    Parameters

    • func: Function

      Functions that require debounce.(需要防抖的函数)

    • delay: number = 300

      Delay，default 300ms.(延迟时间，默认300ms)

    • isImmediate: boolean = false

      Whether to execute the function immediately,Default false.(是否立即执行函数，默认否)

    Returns Function

    Example

    let num=0
    function add(nv){
       num+=nv
    }
    const debounceAdd=PerformanceTool.debounce(add);
    debounceAdd(1);
    debounceAdd(1);
    debounceAdd(1);
    console.info(num); // 1

Static memoize

  • memoize(func, options?): Function

  • Caching function calculation results.(缓存函数的计算结果)

    Parameters

    • func: Function

      Functions that require cache.(需要缓存的函数)

    • options: MemoizeOptions = ...

      An object specifying the caching options.(指定缓存选项的对象)

      • maxCacheSize: The maximum number of cached items allowed. Defaults to Infinity. (缓存的最大数量。默认为 Infinity)
      • expirationTime: The time in milliseconds after which a cached item expires. Defaults to Infinity. (缓存项过期时间，以毫秒为单位。默认为 Infinity)

    Returns Function

    Example

    let count=0
    function addCount(nv){
       count+=nv;
       return count 
    }
    const memoizeAdd=PerformanceTool.memoize(addCount,{
       expirationTime:2000
    })
    memoizeAdd(1);memoizeAdd(1);memoizeAdd(1);
    console.info(count)  //1
    memoizeAdd(2);memoizeAdd(2);memoizeAdd(2);
    console.info(count)  //3
