## 安装

**npm:**

```
npm install boots-js
```

**yarn:**

```
yarn add boots-js
```

**cdn:**

```
https://unpkg.com/boots-js@latest/umd/index.js
```

## 使用

### CommonJS

```javascript
// 全局导入
const BootsJS=require('boots-js/cjs/index');
BootsJS.ObjectUtil.type(123);
// 按需导入
const  ObjectUtil  = require('boots-js/cjs/object-util');
ObjectUtil.type(123);
```
